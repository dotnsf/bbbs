# BBBS

Blockchain-ed BBS Application

## Overview

Good old days BBS with Blockchain technology.

## Pre-requisites

- Node.js V6.x

- BBBS API

## Setup

Edit settings.js:

`exports.api_url = 'http://localhost:6012/api';`


## Install

- $ npm install

- $ node app
